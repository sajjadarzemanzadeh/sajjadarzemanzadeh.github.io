Nonlinear System Identification of Soft Robot Dynamics

Mar. 2021 – July 2021

Instructor: Dr. MRH. Yazdi

The dynamic of a soft robotic actuatoe was estimated using data-driven methods [data source]. Due to nonlinearities in the properties of the soft materials, we cannot use the common linear system identification methods. Koopman operator theory is based on the fact that every nonlinear system has a linear reoresentation in the infinite-dimensional space. Koopman operator algorithm was applied on the data of a soft robotic arm, where the inputs of the model are the control input of three pressure regulators, and output is the end effector position. Besides, LSTM neural network was also utilized to model the soft robot. Time series of input voltage were given to the model to estimate the position of end effector. Trained LSTM model was imported into simulink environment to design a NN controller.
